# Od-Hul Website — Danny Nissani, Attorney

## Project Overview
Legal services website for Danny Nissani, Attorney. Bilingual — English and Hebrew.

## Tech Stack
- Plain HTML + CSS (no framework)
- No build step required

## Repository
- GitHub: asets-gobizit/od-hul-website
- Default branch: main
- Deployed via GitHub Actions → GitHub Pages

## Structure
- index.html, about.html, contact.html, services.html — English pages
- he/ — Hebrew versions of all pages
- logo.png, iconlogo.png — site logos

## Key Notes
- When editing content, update BOTH the English and Hebrew versions
- No npm, no build step — changes to HTML files deploy directly
